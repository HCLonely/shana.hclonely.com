banner

logo

网站 logo

  • expanded: 侧边栏展开时左上角的 logo
  • collapsed: 侧边栏收起时左上角的 logo
  • dark: 顶栏为暗色时左上角的 logo, 仅 about 页面生效

flag

语言标识

  • icon: 语言图标，默认仅有flag-cn和flag-us, 其他图标自行寻找存放于主题目录/source/images/flags/
  • name: 语言名称

示例：

githubCorner

右上角的 github corner

示例：

since

建站年份，显示在页面底部

示例：

menu

[主要]侧边栏菜单设置

  • name: 分组名
  • icon: 分组图标
  • config: [主要]分组内容(详细设置查看Config)，如果有二级菜单则不需要此项！
  • submenu: 二级菜单，如果有二级菜单则不需要config, 此选项内容包含name, icon, config选项

示例：

about

侧边栏的关于本站

  • url: 关于页面链接
  • name: 在侧边栏显示的文字
  • icon: 图标

示例：

aboutPage

关于页面设置

  1. 生成关于页面
  1. 编辑source/about/index.md, 添加type: 'about'
  1. 编辑主题配置文件的aboutPage
  • website: 关于本站内容
    • head: 标题
    • html: 内容，支持html语法
  • webmaster: 关于站长内容
    • head: 标题
    • name: 站长名字
    • url: 链接
    • img: 头像
    • description: 描述
    • html: 其他内容，支持html语法

示例：
